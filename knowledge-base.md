# 조쉬의 콘텐츠 마스터클래스 - AI 조교 지식 베이스

## 강의 개요
콘텐츠 기획, 릴스 편집, AI 활용, 인스타그램 운영을 다루는 17강 마스터클래스입니다.

## 강의별 핵심 내용
- 1강: 콘텐츠 기획의 본질은 만드는 것이 아닌 문제 해결이다
- 2강: 페르소나를 넘어선 페인 포인트(Pain Point) 분석법
- 3강: 절대 마르지 않는 콘텐츠 아이디어 뽑는 법
- 4강: 팔리는 콘텐츠를 만드는 3단계 기획 프레임워크
- 5강: 고객을 멈추게 만드는 후킹 기획의 기술
- 6강: 콘텐츠는 크게 2가지로 나뉜다 (정보성 vs 브랜딩)
- 7강: 사람의 심리를 이용한 7가지 콘텐츠 기획법
- 8강: 조회수는 우연, 전환은 설계. 콘텐츠 기획의 끝은 매출이다
- 9강: 콘텐츠의 조회수는 정해져 있다 - 표본의 법칙
- 10강: 편집 왕초보도 10분 만에 릴스 만들기
- 11강: 캡컷 릴스 편집의 기본 - 30분이면 프로처럼 편집하기
- 12강: 조회수 폭발하는 릴스 편집 공식 - 3초 법칙과 리텐션 편집
- 13강: 콘텐츠 유형별 편집 템플릿 - 따라만 하면 되는 편집 레시피
- 14강: 100만 바이럴 릴스의 공통점 - 성공하는 편집 패턴 분석
- 15강: AI를 활용한 콘텐츠 제작 시간 10배 효율화 방법
- 16강: AI를 활용한 레퍼런스 분석 및 100만 콘텐츠 기획법
- 17강: 실제 매출을 키워주는 찐팬을 만드는 인스타그램 운영 전략

## 조쉬의 핵심 철학
- 콘텐츠는 예술이 아니라 문제 해결이다
- 조회수보다 전환(매출)이 중요하다
- 좋은 콘텐츠는 기획에서 80%가 결정된다
- AI는 도구일 뿐, 기획력이 본질이다
